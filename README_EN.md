# kurohelper-db

`kurohelper-db` is the core database module of the KuroHelper project, responsible for storing and managing part of the data required by KuroHelper.

## Key Features

- Data Access: Provides simple data access methods, making it easy to use in Golang projects.
- Database Support: Currently supports PostgreSQL, with future extensions in mind.

## Installation

> [!IMPORTANT]
> The final version of this project cached by the Go Proxy is **v3.1.0**.
> Subsequent versions require manual building. Please clone the repository and use replace in your go.mod.

Using Go Modules:

- v3(Up to **v3.1.0** only)

```bash
go get github.com/kuro-helper/kurohelper-db/v3
```

- v2

```bash
go get github.com/peter910820/kurohelper-db/v2
```

- v1

```bash
go get github.com/peter910820/kurohelper-db
```
