# db

`db` is the core database module of the KuroHelper project, responsible for storing and managing part of the data required by KuroHelper.

## Key Features

- Data Access: Provides simple data access methods, making it easy to use in Golang projects.
- Database Support: Currently supports PostgreSQL, with future extensions in mind.

## Installation

Using Go Modules:

```bash
go get github.com/kuro-helper/db
```
